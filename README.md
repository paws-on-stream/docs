# Paws on Stream Docs

Dokumentations-Repo für **Paws on Stream**.

Das Repository ist für GitHub Pages vorbereitet und enthält die Doku für:
- **Bot** (`paws_on_stream_bot`)
- **Web** (`paws_on_stream_web`)

## Lokale Vorschau

```bash
pip install -r requirements.txt
mkdocs serve
```

## Build

```bash
mkdocs build
```
