# Paws on Stream

Paws on Stream ist ein Telegram-basiertes Event- und Moderationssystem.

Es besteht aus drei Hauptteilen:

- **Bot**: nimmt Telegram-Nachrichten entgegen, validiert sie und sendet sie an das Backend.
- **Display**: zeigt freigegebene Nachrichten auf dem Pi-Overlay an und liefert Display-Feedback ans Backend.
- **Web**: verwaltet Teilnehmer, Events, Nachrichten, Moderation und Darstellung.

Diese Doku beschreibt den aktuellen Stand von Bot, Display und Web und ist als Grundlage für GitHub Pages gedacht.
