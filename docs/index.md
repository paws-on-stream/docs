# Paws on Stream

Paws on Stream ist ein Telegram-basiertes Event- und Moderationssystem.

Es besteht aus zwei Hauptteilen:

- **Bot**: nimmt Telegram-Nachrichten entgegen, validiert sie und sendet sie an das Backend.
- **Web**: verwaltet Teilnehmer, Events, Nachrichten, Moderation und Darstellung.

Diese Doku beschreibt den aktuellen Stand von Bot und Web und ist als Grundlage für GitHub Pages gedacht.
